## 🌟 Overview

NEON RUNNER is a browser-based endless runner game built using HTML, CSS, and JavaScript. Set in a futuristic cyberpunk environment, players must dodge obstacles, collect coins, and survive for as long as possible while chasing high scores.

---

## ✨ Features

* 🎮 Endless runner gameplay
* 🕹️ Keyboard controls (WASD / Arrow Keys)
* 📱 Mobile swipe controls
* 🪙 Coin collection system
* 📈 Dynamic score tracking
* 🏆 Local leaderboard support
* 🌃 Cyberpunk-inspired visuals
* ⚡ Smooth Canvas-based rendering
* 🎯 Progressive gameplay difficulty

| Canvas API    | Rendering & Animation |

## 🎯 Gameplay

* Move between lanes to avoid obstacles.
* Jump over barriers and slide under incoming hazards.
* Collect data coins to boost your score.
* Survive longer as the game gradually increases in speed.
* Challenge yourself to beat your personal best.
